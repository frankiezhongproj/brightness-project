# Screen Brightness Automation (Python)

## Description
This Python project automatically adjusts screen brightness based on the time of day.

## Features
- Automatically adjusts brightness based on current time of the user
- Uses conditions based on time (morning, afternoon, evening, night)
- Runs using a continuous check loop

## Technologies Used
- Python
- datetime module
- screen_brightness_control library

## How to Run
1. Install required library
2. Run the script


## How It Works
- The program checks current time on your system
- Based on time ranges (morning, afternoon, evening, night), it sets brightness levels
- Uses `screen_brightness_control` to apply changes to the monitor
